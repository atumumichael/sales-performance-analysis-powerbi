# Sales Performance Analysis Dashboard

## 📊 Overview
This project analyzes sales data to track revenue, profit, and performance across regions and product categories.

## 🎯 Objective
To identify key business trends and support data-driven decision-making.

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Key Metrics
- Total Revenue: $1.35M
- Total Orders: 600
- Total Profit: $281K
- Profit Margin: 21.37%

## 📸 Dashboard Preview

## 🔍 Key Insights
- Technology category generated the highest revenue
- North and West regions led in sales performance
- Sales show a declining trend towards year-end
- Top-selling product category includes Chairs and Laptops

## 💡 Business Impact
- Helps identify high-performing regions for expansion
- Supports inventory planning based on product demand
- Highlights revenue trends for strategic decisions
